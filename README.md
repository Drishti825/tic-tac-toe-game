# 🎮 Tic Tac Toe Game

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- 2 Player Game (X vs O)
- Winner Detection
- Draw Detection
- Restart Button
- Interactive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
